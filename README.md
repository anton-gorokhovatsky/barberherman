# barberherman

Независимая адаптивная версия сайта [barberherman.ru](https://barberherman.ru/), реконструированная без зависимости от Readymag.

## Локальный запуск

Сайт статический и не требует сборки:

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080`.

## Публикация

Проект подготовлен для GitHub Pages. Цвета вынесены в CSS-переменные для последующего добавления тёмной темы.
