# barberherman

Независимая статическая копия сайта [barberherman.ru](https://barberherman.ru/), визуально воспроизводящая исходный макет Readymag на десктопе и мобильных устройствах.

## Локальный запуск

Сайт статический и не требует сборки:

```bash
python3 -m http.server 8080
```

Откройте `http://localhost:8080`.

## Публикация

Проект подготовлен для GitHub Pages. Пользовательский домен пока не включён, поэтому оригинальный сайт продолжает работать независимо от этой публикации.
